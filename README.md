## المحتويات

- `app.py`: تشغيل المشروع عبر CLI بدون Gradio.
- `NAMAA_Colab_T4.ipynb`: نوتبوك Colab مجهز للعمل مع GPU T4.
- `requirements.txt`: الاعتمادات المثبتة بإصدارات محددة.


## التشغيل على Colab

1. افتح النوتبوك من الزر بالأعلى.
2. اختر Runtime نوع GPU.
3. شغل الخلايا بالترتيب.
4. راجع ملفات التتبع داخل مجلد `logs/`.

## ملاحظات

- النوتبوك يحتوي نظام Logs مستقل لكل خلية.
- عند إعادة تشغيل أي خلية، يتم حذف ملف الـLog القديم الخاص بها تلقائيًا.
